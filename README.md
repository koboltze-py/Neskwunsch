#  DRK Dienstwünsche - Schichtplanungs-App

Eine Webanwendung für das Deutsche Rote Kreuz Köln zur Verwaltung von Dienstwünschen für die Erste-Hilfe-Station am Flughafen.

##  Features

-  **Benutzer-Authentifizierung** - Login/Registrierung mit Session-Management
-  **Dienstwunsch-System** - Mitarbeiter können Schichtwünsche (Früh/Spät/Nacht) abgeben
-  **Admin-Dashboard** - Übersicht aller Wünsche für Schichtkoordinatoren
-  **Responsive Design** - Funktioniert auf Desktop, Tablet und Smartphone
-  **Datenbankpersistenz** - PostgreSQL (Produktion) / SQLite (Entwicklung)

##  Schnellstart

### Backend starten
```powershell
pip install -r requirements.txt
python app.py
```

### Frontend starten
```powershell
npm install
npm run dev
```
